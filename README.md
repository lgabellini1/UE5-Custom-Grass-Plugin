# Custom procedural grass for Unreal Engine 5
A plugin hooking directly into UE5 rendering pipeline providing a custom, procedural grass system for landscapes.

<em>Note: this is not meant as a publicly available, usable plugin. It is far from being complete or stable enough for such use. It is mostly a personal project for learning and showcase in a portfolio.</em>

# Info

The project was developed for the Unreal Engine version 5.7 on Windows 11.
